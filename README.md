### Quick Setup (you)
1. Replace every `YOUR_USERNAME` and `YOUR_NAME` in this README with your GitHub username & display name.
2. Create repo secrets: `METRICS_TOKEN` (classic PAT with `public_repo, read:user` scopes) and `WAKATIME_API_KEY` (copy from WakaTime > Settings > API Key).
3. Push to a public repo named exactly `YOUR_USERNAME` so GitHub shows it as your profile README.
4. Enable GitHub Actions on the repo. First runs may take a minute; images populate after workflows finish.

### What updates automatically
- `metrics.svg` refreshed daily with contributions, stars, top languages, and activity.
- WakaTime block refreshes daily; first run will replace the placeholder.
- External stat cards and activity graph are live-rendered on each page view (no cron needed).

> Tip: Swap the ASCII art with any file in `svg/`; just paste its contents inside the `<pre>` block.
